# STAR RANKER SYSTEM MANIFEST (v1.0-STABLE)

This document serves as a comprehensive technical and functional knowledge base for the Star Ranker platform.

## 1. System Overview
Star Ranker is a decentralized-style "Oracle Network" and prediction market platform where users influence and predict the relative rankings of real-world entities (Crypto, Technology, Consumer Goods). It combines community sentiment (Voting) with financial forecasting (Staking).

---

## 2. Core Functional Pillars

### A. The Oracle Network (Market Mechanics)
- **Dynamic Ranking**: Items in a category are ranked based on a combination of **Signal Score** (community votes) and **Market Velocity** (staking activity).
- **Settlement Cycles**: Markets "settle" every 30 minutes. Rankings are compared against user predictions to calculate payouts.
- **Influence Levels**: User "Tiers" (Bronze → Diamond) determine the weight of their votes and the maximum stake limits allowed.

### B. Dual-Mode Architecture
- **Live Mode**: Uses real capital for stakes. Payouts are in real currency.
- **Practice Mode (Stars ★)**: A high-fidelity simulation environment.
  - Initial Balance: ★50,000 (Stars).
  - Synchronized Odds: Uses the same math as Live Mode.
  - Risk-Free Training: Allows users to refine "Oracle Strategies" without capital at risk.
- **Visual Branding**: Practice mode is identified by amber accents and the Star icon (★).

### C. Financial Infrastructure
- **Multi-Currency Support**: Real-time conversion between USD, EUR, GBP, JPY, NGN, etc.
- **Sync Ledger**: The "Sync" system ensures the local UI balance is always consistent with the server-side database.
- **Platform Fee**: A standard 5% fee is applied to all stakes (both Live and Practice) to maintain the network.
